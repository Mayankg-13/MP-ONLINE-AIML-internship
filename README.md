# MP ONLINE AIML Internship Assignments

This repository contains the assignments completed during my ongoing AIML Internship at MP ONLINE.

## 📁 Assignments Index

1. **[Adult Census Income Classification](./Adult_Census_Income_Classification)**
   - **Description**: Predict whether an individual's annual income exceeds $50,000 using demographic and employment-related data.
   - **Models Used**: Logistic Regression, Decision Tree, Random Forest, K-Nearest Neighbors (KNN), and Support Vector Machine (SVM).
   - **Key Features**: Comprehensive data cleaning, feature engineering, standard scaling, one-hot encoding, and a comparative evaluation using Accuracy, Precision, Recall, F1-Score, and ROC-AUC.
   - **Deliverables**: Detailed Jupyter Notebook, automated modular python script, performance metrics, and plots of confusion matrices & ROC curves.

2. **[Face Recognition CNN](./Face_Recognition_CNN)**
   - **Description**: Identify individuals from their facial photographs using a Convolutional Neural Network (CNN).
   - **Models Used**: Custom Convolutional Neural Network (CNN) with Conv2D, MaxPooling2D, Dropout, and Dense layers.
   - **Key Features**: Labeled Faces in the Wild (LFW) dataset fetching, pixel standardization, stratified train/test split, convolutional feature extraction, dropout regularization, and diagnostic plotting.
   - **Deliverables**: Detailed executed Jupyter Notebook, automated Python script, model evaluation metrics (Accuracy, Precision, Recall, F1-Score), and evaluation plots (training loss/accuracy curves, confusion matrix heatmap, color-coded sample predictions preview).

3. **[Simple Image Classification MNIST](./Simple_Image_Classification_MNIST)**
   - **Description**: Classify handwritten digits (0-9) using a lightweight Convolutional Neural Network (CNN) on the MNIST dataset.
   - **Models Used**: Custom Convolutional Neural Network (CNN) with Conv2D, MaxPooling2D, Dropout, and Dense layers.
   - **Key Features**: MNIST dataset loading and feature normalization, channel expansion, custom feature extraction, dropout regularization to prevent overfitting, and inline metric logging.
   - **Deliverables**: Detailed executed Jupyter Notebook, modular automated Python script, model evaluation metrics (Accuracy, Precision, Recall, F1-Score), and evaluation plots (training loss/accuracy history, confusion matrix heatmap, color-coded grid predictions preview).

4. **[Cancer Detection CNN](./Cancer_Detection_CNN)**
   - **Description**: Classify histopathology-like cell patch images into Normal vs. Malignant/Cancerous using a Convolutional Neural Network (CNN).
   - **Models Used**: Custom 3-block Convolutional Neural Network (CNN) with Conv2D, MaxPooling2D, Dropout, and Dense layers with a Sigmoid output.
   - **Key Features**: Self-contained H&E-stained Synthetic Cell Patch Generator (simulating pink background cytoplasm, normal spaced light purple nuclei vs. dense irregular hyperchromatic cancer nuclei), stratified train/test split, binary crossentropy loss, and prediction visualization.
   - **Deliverables**: Detailed executed Jupyter Notebook, modular automated Python script, model evaluation metrics (Accuracy, Precision, Recall, F1-Score), and evaluation plots (training curves, confusion matrix, and visual cell predictions with probability scores).
